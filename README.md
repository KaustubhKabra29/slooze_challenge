# Slooze Challenge

## Crawler Features

| Feature           | Description                                       |
|-------------------|---------------------------------------------------|
| Multi-threading    | Supports multiple threads to enhance performance. |
| Data Storage       | Enables saving crawled data in various formats.  |
| Real-time updates   | Fetches and processes updates in real time.      |
| Customizable        | Options available for tailored crawling.         |

## EDA Outputs

| Type        | Description                                     |
|-------------|-------------------------------------------------|
| Graphs      | Visual representations of data relationships.    |
| Statistics  | Key metrics and calculations derived from data.  |
| Data Logs   | Records of data processing and outputs.          | 

## Installation & Usage

Follow these steps for installation:

1. Clone the repo: `git clone https://github.com/KaustubhKabra29/slooze_challenge.git`
2. Navigate to the directory.
3. Install dependencies: `pip install -r requirements.txt`
4. Run the crawler: `python crawler.py`
